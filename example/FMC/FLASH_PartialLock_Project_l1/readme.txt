@par Example Description:

This example demonstrates how to use an algorithm function list within the project.

Flow:
1. Import the function list file ("calculate_symbol*.*") into the IDE project under User Groups.
2. Declare the algorithm functions as global symbols.
3. Call the functions directly in the application as needed.

@par Directory Contents:

- FMC/FLASH_PartialLock_Project_l1/main.c               Main program
- FMC/FLASH_PartialLock_Project_l1/ht32fxxxxx_nn_it.c   Interrupt handlers
- FMC/FLASH_PartialLock_Project_l1/calculate_symbol_*.* Algorithm function list

@par Hardware and Software Environment:

- This example can be run on HT32 Series development board.
